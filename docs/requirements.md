# Personalized Career Orientation & Learning Roadmap Platform

## Overview

A web platform that helps Software Engineering students identify suitable career paths, analyze skill gaps, receive AI-powered career guidance, and build a professional portfolio.

## Technology Stack

### Backend
- ASP.NET Core 8 Web API
- Entity Framework Core 8
- SQL Server
- JWT Authentication
- AutoMapper
- FluentValidation

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Axios

### AI
- OpenAI API

### External Integration
- GitHub API

---

## User Roles

### Student
- Register/Login
- Manage Profile
- Select Career Path
- View Roadmap
- Track Progress
- Analyze Skill Gap
- Chat with AI Mentor
- Manage Portfolio

### Admin
- Manage Users
- Manage Career Paths
- Manage Skill Nodes
- View Dashboard Statistics

### Authentication
